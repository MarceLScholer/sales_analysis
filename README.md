# Sales Analysis – Python & Pandas

Exploratory data analysis of a retail sales dataset using Python, Pandas, and Matplotlib.

## Overview
This project analyzes sales performance across three dimensions:
- **Region** – Central, Este, Oeste
- **Product** – Sistemas, Dispositivos, Accesorios
- **Salesperson** – individual performance comparison

## Key Findings
- Total sales: **$239,533**
- The **Central** region leads with $90,295 (38% of total)
- **Sistemas** is the best-selling category with $133,642 (56% of total)
- **Karen** is the top salesperson with $50,490, followed by Kevin with $48,310

## Tools Used
- Python 3
- Pandas
- Matplotlib
- Jupyter Notebook
